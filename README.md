# CampusOps Admin

A complete, production-ready Admin Panel for managing faculty tasks, meetings, workload, and reports in a college.

## Tech Stack

- **Frontend**: React + Vite + TailwindCSS
- **Backend**: Node.js + Express
- **Database**: MongoDB
- **Authentication**: JWT (Admin only)

## Features

- **Authentication**: Admin login, secure sessions.
- **Dashboard**: Statistics, charts (Task status, Availability).
- **Faculty Management**: CRUD, Search, Profiles.
- **Task Management**: Lifecycle (Assigned -> Verified), filtering.
- **Meeting Management**: Scheduling, Attendance.
- **Daily Planner**: Conflicts detection.
- **Leave Management**: Approvals, Substitution.
- **Reports**: PDF exports, Analytics.
- **Notifications**: Broadcasts, Reminders.
- **Settings**: System configuration.

## Setup

### Backend
1. Navigate to `server`
2. `npm install`
3. `npm run dev`

### Frontend
1. Navigate to `client`
2. `npm install`
3. `npm run dev`
